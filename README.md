NotePad-Android应用的介绍文档

一.初始应用的功能

1.新建笔记和编辑笔记

(1)在主界面点击右下方蓝色加号所示按钮，新建笔记并进入编辑界面



(2)进入笔记编辑界面后，可进行笔记编辑

2.笔记列表

在进行笔记的新建和编辑后，在主界面中呈现笔记列表。在初始应用中背景颜色为黑色，字体颜色为白色，且笔记列表中的每个条目都只显示笔记标题

1.功能要求

1.功能要求
